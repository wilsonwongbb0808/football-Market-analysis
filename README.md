# Football Market Analysis

滚球盘口分析工具，用于整理欧赔、亚盘、大小球、比分节点和资金分配建议。

## 使用

- 打开 `index.html` 即可使用。
- GitHub Pages 发布后，可直接通过网页链接访问。

## 主要功能

- 欧赔核心思维分析：初赔结构、去水概率、降赔真假、平局防范。
- 亚盘确认和修正：以欧赔为主线，亚盘用于验证方向和选择下注载体。
- 大小球和让球路径冲突过滤。
- 推荐看板、推荐比分、资金分配建议。
- 本地 24 小时保存最多 6 场比赛记录。
- 每场比赛独立保存过程节点。

## 目录

- `index.html`：发布入口。
- `football-odds-analyzer.html`：工具原始页面。
- `data/`：回测、校准、Monte Carlo 和模型结果。
- `tools/`：数据抓取、回测和优化脚本。

## 提醒

本工具只用于盘口分析和资金计划参考，不保证命中。
